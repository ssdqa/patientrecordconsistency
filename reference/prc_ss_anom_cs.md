# *Single Site, Anomaly Detection, Cross-Sectional*

*Single Site, Anomaly Detection, Cross-Sectional*

## Usage

``` r
prc_ss_anom_cs(process_output)
```

## Arguments

- process_output:

  the output from `prc_process`

## Value

a bar graph displaying the jaccard similarity index for the two events
within each user defined follow up window
