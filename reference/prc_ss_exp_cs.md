# *Single Site, Exploratory, Cross-Sectional*

*Single Site, Exploratory, Cross-Sectional*

## Usage

``` r
prc_ss_exp_cs(process_output)
```

## Arguments

- process_output:

  the output from `prc_process`

## Value

a bar graph displaying the proportion (left y-axis) and raw count (right
y-axis) of patients who have only event a, only event b, both events, or
neither event present in their record
